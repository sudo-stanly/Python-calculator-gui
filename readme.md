## Tkinter Calculator
Welcome to the **Calculator That Works™** – a Python-powered calculator built with Tkinter.  
It adds, subtracts, multiplies, divides, etc.  

---

## Features
- **Numbers**: all 10 of them.
- **Basic ops**: `+ - x ÷`
- **Other ops**:  
  - `√` → Square root
  - `x²` → Square
  - `mod` → Modulo 
  - `Π` → Pi
- **C button**: Clears the input.  
- **⌫ button**: To erase. 
- **= button**: Getting the value.
